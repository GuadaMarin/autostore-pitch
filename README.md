# Pitch · Simulación AutoStore — Forus S.A. (Grupo 12)

Presentación (pitch) del proyecto **Simulación AutoStore para Forus S.A.**,
desarrollado para el *Taller de Ingeniería de Software* (Junio 2026).

Es una presentación web autónoma (un solo archivo `index.html`) con:

- Fondo de video en bucle de los primeros 19 s del
  [video oficial del AutoStore de Forus S.A.](https://youtu.be/67FhJ_u7A8Y).
- 10 slides con el arco del pitch: problema, solución, arquitectura, KPIs,
  alcance, impacto y equipo.
- Navegación por teclado (`←` `→` `Espacio`, `Inicio`/`Fin`, `F` pantalla completa),
  botones, puntos laterales y soporte táctil.

## Ver la presentación

En línea (GitHub Pages): se publica automáticamente al hacer push a `main`.

En local, **no** abras el archivo con doble clic (el embed de YouTube se bloquea
en `file://`). Levanta un servidor local:

```bash
python -m http.server 8123
```

Luego abre <http://localhost:8123/index.html> y pulsa `F` para pantalla completa.
